How It Works

ESP32 to Dashboard in 4 Steps

📡

1. Sense

ESP32 reads temperature, humidity and soil moisture every 30 seconds

📤

2. Send

Data is sent via HTTP POST to the Flask backend API over WiFi

🖥️

3. Process

Server stores data in SQLite DB and evaluates spray conditions

🌊

4. Spray

Servo motor activates the nozzle based on automated or manual commands

About CropCare

Intelligent Agriculture
for Modern Farmers

CropCare is a student IoT project aimed at reducing pesticide waste and improving crop yield through automated, sensor-driven spray control systems.

Why CropCare?

Traditional manual pesticide spraying wastes significant resources — farmers often over-spray or spray at the wrong times, leading to chemical runoff, increased costs, and potential crop damage.

CropCare uses an ESP32 microcontroller with DHT22 temperature/humidity sensors and capacitive soil moisture sensors to continuously monitor field conditions. When the system detects unfavorable conditions, it triggers a servo-controlled nozzle to apply pesticide precisely.

Our web dashboard gives farmers full visibility and control — from anywhere in the world, on any device.
